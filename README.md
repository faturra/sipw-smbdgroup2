# Usage

* Run services in the background:
`docker-compose up -d`

* Or run services in the foreground:
`docker-compose up --build`

* Inspect volume:
`docker volume ls`
and
`docker volume inspect <volume name>`

* View networks:
`docker network ls`

* Bring services down:
`docker-compose down`
