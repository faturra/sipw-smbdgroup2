# Usage

* Run services :
`docker-compose up`
or run service in the backgrond
`docker-compose up -d`

* Run services in the foreground:
`docker-compose up --build`

* Inspect volume:
`docker volume ls`
and
`docker volume inspect <volume name>`

* View networks:
`docker network ls`

* Bring services down:
`docker-compose down`
